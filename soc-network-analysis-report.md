Introduction:

&nbsp;           

&nbsp;           This project focuses on understanding network behavior and identifying potential security threats using cybersecurity tools. The project simulates basic Security Operations Center (SOC) activities such as network scanning, traffic monitoring, and analyzing suspicious activities in a controlled lab environment.



Objective:

* To understand how network scanning works
* To analyze network traffic
* To identify suspicious activities in a network
* To gain practical experience with cybersecurity tools


Tools Used:   

* Kali Linux



* Nmap



* Wireshark


Description:

* Kali Linux was used as the testing environment.



* Nmap was used for network scanning and host discovery.



* Wireshark was used to capture and analyze network packets.


Methodology:

* Setup a cybersecurity lab environment using Kali Linux.



* Identified the target system in the network.



* Performed network scanning using Nmap.



* Captured network traffic using Wireshark.



* Analyzed packets to identify suspicious activities.


Results:

&nbsp;       The network scan identified open ports and running services on the target machine. Packet analysis revealed different types of network communication such as HTTP and DNS requests. This helped understand how attackers may identify vulnerable services in a network.


Conclusion:

&nbsp;          This project helped in understanding basic cybersecurity concepts such as network scanning, packet analysis, and threat detection. It provided practical exposure to tools commonly used by SOC analysts and security professionals.
