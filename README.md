# Network Scanning and Traffic Analysis Project

## Overview

This project demonstrates basic cybersecurity monitoring and analysis techniques used in a Security Operations Center (SOC). The focus of this project is to understand network behavior, identify open ports, and analyze network traffic to detect potential security risks.

## Objective

The objective of this project is to gain practical knowledge of network scanning and packet analysis using industry-standard cybersecurity tools.

## Tools Used

* Kali Linux – Security testing environment
* Nmap – Network scanning and port discovery
* Wireshark – Network packet capture and analysis

## Methodology

1. Set up a testing environment using Kali Linux.
2. Performed network scanning using Nmap to identify active hosts and open ports.
3. Captured network traffic using Wireshark.
4. Analyzed captured packets to understand communication protocols and traffic patterns.

## Results

The network scan successfully identified open ports and running services on the target system. Packet analysis revealed different network protocols such as TCP, DNS, and HTTP traffic. This helped in understanding how attackers may gather information about systems in a network.
